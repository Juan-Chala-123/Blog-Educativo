Etiquetas de Formularios HTML

Aprende a crear formularios mostrando su código y resultado.

Input Text

¿Qué es? Permite al usuario escribir texto libre, como nombres o datos cortos.

Campo para ingresar texto corto.

<label>Nombre</label>
<input type="text" placeholder="Escriba su nombre">

Input Email

¿Qué es? Campo para ingresar una dirección de correo electrónico, valida el formato automáticamente.

<label>Email</label>
<input type="email" placeholder="correo@empresa.com">

Password

¿Qué es? Campo para ingresar contraseñas, oculta el texto escrito por seguridad.

<label>Contraseña</label>
<input type="password">

Number

¿Qué es? Permite ingresar solo valores numéricos, útil para cantidades, edades o identificaciones.

<label>Identificación</label>
<input type="number">

Date

¿Qué es? Permite seleccionar una fecha desde un calendario emergente.

<label>Fecha</label>
<input type="date">

Radio

¿Qué es? Permite seleccionar una sola opción entre varias alternativas.


Masculino Femenino 
<input type="radio" name="sexo"> Masculino
<input type="radio" name="sexo"> Femenino

Select

¿Qué es? Muestra una lista desplegable para elegir una opción entre varias.

<select>
<option>Soltero</option>
<option>Casado</option>
</select>

Checkbox

¿Qué es? Permite seleccionar una o varias opciones de una lista.


Técnico Tecnólogo Especialización 
<input type="checkbox"> Técnico
<input type="checkbox"> Tecnólogo
<input type="checkbox"> Especialización

File

¿Qué es? Permite al usuario seleccionar y subir archivos desde su dispositivo.

<input type="file">

Textarea

¿Qué es? Área de texto para escribir mensajes largos o comentarios.

<textarea rows="3"></textarea>

Button

¿Qué es? Botón que ejecuta una acción, como enviar el formulario o mostrar un mensaje.

<button>Click</button>, <button onclick="alert('Hola 👋')">Click /button>
