Etiquetas de Formularios HTML

Aprende a crear formularios mostrando su código y resultado.

Input Text

Campo para ingresar texto corto.

<label>Nombre</label> <input type="text" placeholder="Escriba su nombre">

Input Email

<label>Email</label> <input type="email" placeholder="correo@empresa.com">

Password

<label>Contraseña</label> <input type="password">

Number

<label>Identificación</label> <input type="number">

Date

<label>Fecha</label> <input type="date">

Radio


Masculino Femenino
<input type="radio" name="sexo"> Masculino <input type="radio" name="sexo"> Femenino

Select

<select> <option>Soltero</option> <option>Casado</option> </select>

Checkbox


Técnico Tecnólogo Especialización
<input type="checkbox"> Técnico <input type="checkbox"> Tecnólogo <input type="checkbox"> Especialización

File

<input type="file">

Textarea

<textarea rows="3"></textarea>

Button

<button>Click</button>, <button onclick="alert('Hola 👋')">Click /button>
